## 🚀 How to Use

1. Run the tool: `python titan_x_bomber.py`
2. Select option **[1] START ATTACK**
3. Enter target number **(10 digits, without 0 or 880)**
4. Enter message count **(1-100)**
5. Choose speed mode:
   - **Turbo** (0.05s - Fastest)
   - **Normal** (0.4s - Stable)
   - **Safe** (1.0s - Anti-Ban)
6. Wait for completion
